# go-mockserver

## Development Setup

Install & setup [direnv](https://direnv.net/):

```
brew install direnv
echo 'eval "$(direnv hook zsh)"' >> ~/.zshrc
```

Install latest version of [devbox](https://www.jetify.com/devbox/):

```
curl -fsSL https://get.jetify.com/devbox | bash
```

## Usage

```
task compile
task run
```
